Is Configuration Manager Dead

    No, it’s not dead and won’t be for while

    This blog contains various tweets, and other resources to help showcase the continued love and development of the Configuration Manager Platform. The twitter link below will point you to the owner of this site, and the Github link will direct you to the repository so you can make requests to add to his site.

    Configuration Manager 2409 Release

    While this is a bit of an older message, Configuration Manager 2409 has been released as a new build. This build from Configuration Manager now has some security SQL changes, and some CMG updates. This post is largely a test to validate the build process now works as intended.
